# project_mcu

1장 ATmega128

특징 : RISC 구조, 주소지정 모드(addressing mode)제공, ISP기능 등

 - RISC : 명령어 수를 줄여서 한 명령을 빠르게 수행하는 CPU 설계방식

 - ISP : MCU를 보드에 장착한 상태에서 SPI 등을 통해 프로그램을 바로 다운/수정하는 방식

 - SPI : 클럭을 기준으로 마스터-슬레이브가 MOSI/MISO로 고속데이터를 주고받는 동기 직렬 통신방식
